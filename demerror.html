    DEM Error Correction Module¶

    This Python module implements the calculation of correction for residual topographic effects (a.k.a. as DEM errors).

    exception pyrate.core.dem_error.DEMError[source]¶

    Generic exception for DEM errors.

    pyrate.core.dem_error.calc_dem_errors(ifgs: list, bperp: numpy.ndarray, look_angle: numpy.ndarray, range_dist: numpy.ndarray, threshold: int) → Tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray][source]¶

    Function to calculate the DEM error for each pixel using least-squares adjustment of phase data and perpendicular baseline. The least-squares adjustment co-estimates the velocities. - nrows is the number of rows in the ifgs, and - ncols is the number of columns in the ifgs.

    Parameters

    • ifgs – list of interferogram class objects.

    • bperp – Per-pixel perpendicular baseline for each interferogram

    • look_angle – Per-pixel look angle

    • range_dist – Per-pixel range distance measurement

    • threshold – minimum number of redundant phase values at pixel (config parameter de_pthr)

    Returns

    dem_error: estimated per-pixel dem error (nrows x ncols)

    Returns

    dem_error_correction: DEM error correction for each pixel and interferogram (nifgs x nrows x ncols)

    Returns

    vel: velocity estimate for each pixel (nrows x ncols)
