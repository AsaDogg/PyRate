    Atmospheric Phase Screen Module¶

    This Python module implements a spatio-temporal filter method for correcting interferograms for atmospheric phase screen (APS) signals.

    pyrate.core.aps.gaussian_spatial_filter(image: numpy.ndarray, cutoff: float, x_size: float, y_size: float, nanfill: bool = True, fillmethod: str = 'nearest') → numpy.ndarray[source]¶

    Function to apply a Gaussian spatial low-pass filter to a 2D image with unequal pixel resolution in x and y dimensions. Performs filtering in the Fourier domain. Any NaNs in the image are interpolated prior to Fourier transformation, with NaNs being replaced in to the filtered output image. :param image: 2D image to be filtered :param cutoff: filter cutoff in kilometres :param x_size: pixel size in x dimension, in metres :param y_size: pixel size in y dimension, in metres :param nanfill: interpolate image to fill NaNs :param fillmethod: interpolation method (‘nearest’, ‘cubic’, or ‘linear’) :return: filt: Gaussian low-pass filtered 2D image

    pyrate.core.aps.gaussian_temporal_filter(tsincr: numpy.ndarray, cutoff: float, span: numpy.ndarray, thr: int) → numpy.ndarray[source]¶

    Function to apply a Gaussian temporal low-pass filter to a 1D time-series vector for one pixel with irregular temporal sampling. :param tsincr: 1D time-series vector to be filtered. :param cutoff: filter cutoff in years. :param span: 1D vector of cumulative time spans, in years. :param thr: threshold for non-NaN values in tsincr. :return: ts_lp: Low-pass filtered time series vector.

    pyrate.core.aps.spatial_low_pass_filter(ts_hp: numpy.ndarray, ifg: pyrate.core.shared.Ifg, params: dict) → numpy.ndarray[source]¶

    Filter time series data spatially using a Gaussian low-pass filter defined by a cut-off distance. If the cut-off distance is defined as zero in the parameters dictionary then it is calculated for each time step using the pyrate.covariance.cvd_from_phase method. :param ts_hp: Array of temporal high-pass time series data, shape (ifg.shape, n_epochs) :param ifg: pyrate.core.shared.Ifg Class object. :param params: Dictionary of PyRate configuration parameters. :return: ts_lp: Low-pass filtered time series data of shape (ifg.shape, n_epochs).

    pyrate.core.aps.spatio_temporal_filter(params: dict) → None[source]¶

    Applies a spatio-temporal filter to remove the atmospheric phase screen (APS) and saves the corrected interferograms. Firstly the incremental time series is computed using the SVD method, before a cascade of temporal then spatial Gaussian filters is applied. The resulting APS corrections are saved to disc before being subtracted from each interferogram.

    Parameters

    params – Dictionary of PyRate configuration parameters.

    Isolate high-frequency components of time series data by subtracting low-pass components obtained using a Gaussian filter defined by a cut-off time period (in days). :param tsincr: Array of incremental time series data of shape (ifg.shape, n_epochs). :param epochlist: A pyrate.core.shared.EpochList Class instance. :param params: Dictionary of PyRate configuration parameters. :return: ts_hp: Filtered high frequency time series data; shape (ifg.shape, nepochs).
