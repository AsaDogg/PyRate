    Prepifg Helper Module¶

    This Python module converts interferogram input files to a common geotiff format with PyRate specific metadata headers. The module also implements multilooking/downsampling and cropping operations to reduce the size of the computational problem.

    pyrate.core.prepifg_helper.coherence_paths_for(path: str, params: dict, tif=False) → str[source]¶

    Returns path to coherence file for given interferogram. Pattern matches based on epoch in filename.

    Example

    ‘20151025-20160501_eqa_filt.cc’ Date pair is the epoch.

    Parameters

    • path – Path to intergerogram to find coherence file for.

    • params – Parameter dictionary.

    • tif – Find converted tif if True (_cc.tif), else find .cc file.

    Returns

    Path to coherence file.

    Open, resample, crop and optionally save to disk an interferogram or DEM. Returns are only given if write_to_disk=False

    Parameters

    • raster_path (str) – Input raster file path name

    • xlooks (int) – Number of multi-looks in x; 5 is 5 times smaller, 1 is no change

    • ylooks (int) – Number of multi-looks in y

    • exts (tuple) – Tuple of user defined georeferenced extents for new file: (xfirst, yfirst, xlast, ylast)cropping coordinates

    • thresh (float) – NaN fraction threshold below which resampled_data is assigned NaNs

    • crop_opt (int) – Crop option

    • write_to_disk (bool) – Write new data to disk

    • out_path (str) – Path for output file

    • header (dict) – dictionary of metadata from header file

    Returns

    resampled_data: output cropped and resampled image
