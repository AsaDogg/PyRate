    Reference Phase Calculation Module¶

    This Python module implements a reference phase estimation algorithm.

    exception pyrate.core.ref_phs_est.ReferencePhaseError[source]¶

    Generic class for errors in reference phase estimation.

    pyrate.core.ref_phs_est.est_ref_phase_method1(ifg_paths, params)[source]¶

    Reference phase estimation using method 1. Reference phase is the median of the whole interferogram image.

    Parameters

    • ifg_paths (list) – List of interferogram paths or objects

    • params (dict) – Dictionary of configuration parameters

    Returns

    ref_phs: Numpy array of reference phase values of size (nifgs, 1)

    Return type

    ndarray

    Returns

    ifgs: Reference phase data is removed interferograms in place

    pyrate.core.ref_phs_est.est_ref_phase_method2(ifg_paths, params, refpx, refpy)[source]¶

    Reference phase estimation using method 2. Reference phase is the median calculated with a patch around the supplied reference pixel.

    Parameters

    • ifg_paths (list) – List of interferogram paths or objects.

    • params (dict) – Dictionary of configuration parameters

    • refpx (int) – Reference pixel X found by ref pixel method

    • refpy (int) – Reference pixel Y found by ref pixel method

    Returns

    ref_phs: Numpy array of reference phase values of size (nifgs, 1)

    Return type

    ndarray

    Returns

    ifgs: Reference phase data is removed interferograms in place
