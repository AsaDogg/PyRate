    Minimum Spanning Tree Module¶

    This Python module implements the minimum spanning tree functionality for selecting interferometric observations.

    pyrate.mst.mst_boolean_array(ifgs)[source]¶

    Returns a 3D array of booleans constituting valid interferogram connections in the Minimum Spanning Tree matrix.

    Parameters

    ifgs (list) – Sequence of interferogram objects

    Returns

    result: Array of booleans representing valid ifg connections

    Return type

    ndarray

    pyrate.mst.mst_from_ifgs(ifgs)[source]¶

    Returns a Minimum Spanning Tree (MST) network from the given interferograms using Kruskal’s algorithm. The MST is calculated using a weighting based on the number of NaN cells in the phase band.

    Parameters

    ifgs (list) – List of interferogram objects (Ifg class)

    Returns

    edges: The number of network connections

    Return type

    int

    Returns

    is_tree: A boolean that is True if network is a tree

    Return type

    bool

    Returns

    ntrees: The number of disconnected trees

    Return type

    int

    Returns

    mst_ifgs: Minimum Spanning Tree network of interferograms

    Return type

    list
