    This Python module runs the main PyRate processing workflow

    pyrate.scripts.run_pyrate.process_ifgs(ifg_paths, params, rows, cols)[source]¶

    Top level function to perform PyRate workflow on given interferograms

    Parameters

    • ifg_paths (list) – List of interferogram paths

    • params (dict) – Dictionary of configuration parameters

    • rows (int) – Number of sub-tiles in y direction

    • cols (int) – Number of sub-tiles in x direction

    Returns

    refpt: tuple of reference pixel x and y position

    Return type

    tuple

    Returns

    maxvar: array of maximum variance values of interferograms

    Return type

    ndarray

    Returns

    vcmt: Variance-covariance matrix array
