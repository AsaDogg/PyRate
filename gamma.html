    GAMMA Module¶

    This Python module contains tools for reading GAMMA format input data.

    exception pyrate.core.gamma.GammaException[source]¶

    Gamma generic exception class

    pyrate.core.gamma.baseline_paths_for(path: str, params: dict) → str[source]¶

    Returns path to baseline file for given interferogram. Pattern matches based on epoch in filename.

    Example

    ‘20151025-20160501_base.par’ Date pair is the epoch.

    Parameters

    • path – Path to intergerogram to find baseline file for.

    • params – Parameter dictionary.

    • tif – Find converted tif if True (_cc.tif), else find .cc file.

    Returns

    Path to baseline file.

    pyrate.core.gamma.combine_headers(hdr0, hdr1, dem_hdr, base_hdr=None)[source]¶

    Combines metadata for first and second image epochs, DEM and baselines into a single dictionary for an interferogram.

    • hdr0 (dict) – Metadata for the first image

    • hdr1 (dict) – Metadata for the second image

    • dem_hdr (dict) – Metadata for the DEM

    • base_hdr (dict) – Metadata for baselines (if available)

    Returns

    chdr: combined metadata

    Return type

    dict

    pyrate.core.gamma.parse_epoch_header(path)[source]¶

    Returns dictionary of epoch metadata required for PyRate

    Parameters

    path (str) – Full path to GAMMA mli.par file. Note that the mli.par is required as input since the baseline calculations require the input values valid for the GAMMA multi-looked products and also the GAMMA lookup table gives radar coordinates for the multi-looked geometry.

    Returns

    subset: subset of full metadata

    Return type

    dict

    pyrate.core.gamma.read_lookup_table(head, data_path, xlooks, ylooks, xmin, xmax, ymin, ymax)[source]¶

    Creates a copy of input lookup table file in a numpy array and applies the ifg ML factors

    Parameters

    • head (IFG object) – first IFG in the list to read metadata

    • data_path (str) – Input file

    • xlooks (int) – multi-looking factor in x

    • ylooks (int) – multi-looking factor in y

    • xmin (int) – start pixel of cropped extent in x

    • xmax (int) – end pixel of cropped extent in x

    • ymin (int) – start pixel of cropped extent in y

    • ymax (int) – end pixel of cropped extent in y

    Returns

    np-array lt_data_az: azimuth (i.e. row) of radar-coded MLI

    Returns

    np-array lt_data_rg: range (i.e. column) of radar-coded MLI
