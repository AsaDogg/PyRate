    Orbital Error Correction Module¶

    This Python module implements residual orbital corrections for interferograms.

    exception pyrate.orbital.OrbitalError[source]¶

    Generic class for errors in orbital correction.

    pyrate.orbital.get_design_matrix(ifg, degree, offset, scale=100.0)[source]¶

    Returns orbital error design matrix with columns for model parameters.

    Parameters

    • class instance ifg (Ifg) – interferogram to get design matrix for

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • offset (bool) – True to include offset column, otherwise False.

    • scale (float) – Scale factor to divide cell size by in order to improve inversion robustness

    Returns

    dm: design matrix

    Return type

    ndarray

    pyrate.orbital.get_network_design_matrix(ifgs, degree, offset)[source]¶

    Returns larger-format design matrix for network error correction. The network design matrix includes rows which relate to those of NaN cells.

    Parameters

    • ifgs (list) – List of Ifg class objects

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • offset (bool) – True to include offset cols, otherwise False.

    Returns

    netdm: network design matrix

    Return type

    ndarray

    pyrate.orbital.independent_orbital_correction(ifg, degree, offset, params)[source]¶

    Calculates and removes an orbital error surface from a single independent interferogram.

    Warning: This will write orbital error corrected phase_data to the ifg.

    Parameters

    • class instance ifg (Ifg) – the interferogram to be corrected

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • offset (bool) – True to calculate the model using an offset

    • params (dict) – dictionary of configuration parameters

    Returns

    None - interferogram phase data is updated and saved to disk

    pyrate.orbital.network_orbital_correction(ifgs, degree, offset, params, m_ifgs=None, preread_ifgs=None)[source]¶

    This algorithm implements a network inversion to determine orbital corrections for a set of interferograms forming a connected network.

    Warning: This will write orbital error corrected phase_data to the ifgs.

    Parameters

    • ifgs (list) – List of Ifg class objects reduced to a minimum spanning tree network

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • offset (bool) – True to calculate the model using offsets

    • params (dict) – dictionary of configuration parameters

    • m_ifgs (list) – list of multilooked Ifg class objects (sequence must be multilooked versions of ‘ifgs’ arg)

    • preread_ifgs (dict) – Dictionary containing information specifically for MPI jobs (optional)

    Returns

    None - interferogram phase data is updated and saved to disk

    pyrate.orbital.remove_orbital_error(ifgs, params, preread_ifgs=None)[source]¶

    Wrapper function for PyRate orbital error removal functionality.

    NB: the ifg data is modified in situ, rather than create intermediate files. The network method assumes the given ifgs have already been reduced to a minimum spanning tree network.

    Parameters

    • ifgs (list) – List of interferograms class objects

    • params (dict) – Dictionary containing configuration parameters

    • preread_ifgs (dict) – Dictionary containing information specifically for MPI jobs (optional)

    Returns

    None - interferogram phase data is updated and saved to disk
