    Orbital Error Correction Module¶

    This Python module implements residual orbital corrections for interferograms.

    exception pyrate.core.orbital.OrbitalError[source]¶

    Generic class for errors in orbital correction.

    pyrate.core.orbital.calc_network_orb_correction(src_ifgs, degree, scale, nepochs, intercept=False)[source]¶

    Calculate and return coefficients for the network orbital correction model given a set of ifgs: :param list src_ifgs: iterable of Ifg objects :param str degree: the degree of the orbital fit (planar, quadratic or part-cubic) :param int scale: Scale factor for design matrix to improve inversion robustness :param int nepochs: The number of epochs in the network :param intercept: whether to include a constant offset to fit to each ifg. This intercept is discarded and not returned.

    Return coefs

    a list of coefficient lists, indexed by epoch. The coefficient lists are in the following order:

    PLANAR - x, y QUADRATIC - x^2, y^2, x*y, x, y PART_CUBIC - x*y^2, x^2, y^2, x*y, x, y

    pyrate.core.orbital.get_design_matrix(ifg, degree, intercept: Optional[bool] = True, scale: Optional[int] = 1)[source]¶

    Returns orbital error design matrix with columns for model parameters.

    Parameters

    • ifg (Ifg class instance) – interferogram to get design matrix for

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • intercept (bool) – whether to include column for the intercept term.

    • scale (int) – Scale factor for design matrix to improve inversion robustness

    Returns

    dm: design matrix

    Return type

    ndarray

    pyrate.core.orbital.get_network_design_matrix(ifgs, degree, scale, intercept=True)[source]¶

    Returns larger-format design matrix for network error correction. The network design matrix includes rows which relate to those of NaN cells.

    Parameters

    • ifgs (list) – List of Ifg class objects

    • degree (str) – model to fit (PLANAR / QUADRATIC / PART_CUBIC)

    • scale (int) – Scale factor for design matrix to improve inversion robustness

    • intercept (bool) – whether to include columns for intercept estimation.

    Returns

    netdm: network design matrix

    Return type

    ndarray

    pyrate.core.orbital.network_orbital_correction(ifg_paths, params, m_ifgs: Optional[List] = None)[source]¶

    This algorithm implements a network inversion to determine orbital corrections for a set of interferograms forming a connected network.

    Warning: This will write orbital error corrected phase_data to the ifgs.

    Parameters

    • ifg_paths (list) – List of Ifg class objects reduced to a minimum spanning tree network

    • params (dict) – dictionary of configuration parameters

    • m_ifgs (list) – list of multilooked Ifg class objects (sequence must be multilooked versions of ‘ifgs’ arg)

    Returns

    None - interferogram phase data is updated and saved to disk
